UVLT 3問セットCAT 試作版

GitHub Pages 上で動作する、研究用途向けの静的ブラウザ版です。

受験の流れ

  • 対象は UVLT 1k–5k の 5 band です。
  • 1問ずつではなく、3問セットで出題します。
  • 1つのセットの中では、同じ語を重複して選べません。
  • 本番では、解答に応じて 次に出るセット が変わります。
  • 終了時に Excel を自動保存し、必要なら JSON も保存できます。

結果の見方

  • 結果画面では、推定語彙サイズの目安 と 強み・今後伸ばしたい帯域 を表示します。
  • 5つの帯域の解答をまとめて使い、全体の語彙の広がり を推定します。
  • 練習の結果は本番スコアに含めません。
  • 途中の状態はこの端末に一時保存され、24時間を過ぎると自動で削除 します。

受験者情報

保存されるファイル名は 学籍番号_受験者名_日時 を含む形式になります。

Open Science の方針に基づき、問題文・選択肢・正答・尺度情報は公開しています。授業内の成績評価ではなく、研究・学習・試作検証 を主な用途としています。

練習は本番スコアに含まれません

練習・本番の途中では、ブラウザの「戻る」、再読み込み、タブ切り替え、タブ閉じを避けてください。離脱操作はログに記録されます。

練習フィードバック

    受験中はブラウザの「戻る」、再読み込み、他タブへの切り替えを避けてください。離脱操作が起きた場合は警告を出し、結果ログにも残します。
